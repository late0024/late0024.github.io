h1 tag
heading tag, usually bigger than regular paragraph text and is treated as titling
blockquote tag
a tag that is used for quotation or paraphrasing that indents text to its own area
img tag "img src"
a tag that allows adding images to websites through the folder available in the same path as the projects
links tag "a href"
a tag that adds links to the webpage created that can direct you to where the link asks for
cite tag
a tag that allows for writing citations which also applies strong and em tags
